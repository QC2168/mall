# MALL 购物商城项目

### 描述

mall是基于VUE全家桶（Vue2.X、VueRouter、Vuex）、axios、better-scroll 等插件开发的购物商城项目

### 主要技术栈

- vue 2.x
- vue-router
- vuex
- vant-ui
- axios
- better-scroll

### 如何跑起来

**1、下载项目**

```
git clone 
```

**2、进入项目文件夹中开启一个终端，进行依赖安装**

```
npm i
```

**3、启动serve服务**

```
npm run serve
```
